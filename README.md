# Student-Loan-Calculator
A student loan calculator built from a single Claude AI prompt.

**Prompt used:**
> Build a student loan calculator that has inputs for total loan amount, interest rate, and loan term. The results should show a monthly payment and suggest rounding up to the nearest 50 and nearest 100 and show the user what the estimated pay off time would be with that method. This tool should be designed in a modern way with a white background, light gray input fields, san serif fonts, and green button. Centered at the top of the code should be an H1 tag that reads "Student Loan Calculator".
